# 审讯室 · INTERROGATION ROOM

一款沉浸式互动文字冒险游戏，你将扮演审讯官，通过对话与嫌疑人展开心理博弈，揭开案件真相。

## 游戏特色

- **心理博弈系统**：通过观察嫌疑人的紧张度、防御值、信任度变化，判断最佳审讯策略
- **四种审讯语气**：威压、温和、欺骗、逻辑，每种语气对不同性格的嫌疑人效果各异
- **测谎仪机制**：实时显示嫌疑人回答的可信度
- **分支结局**：根据审讯表现，案件可能走向告破或僵局
- **语音输入**：支持 Web Speech API 语音输入，提升沉浸感
- **进度存档**：自动保存审讯进度，随时继续

## 审讯机制

### 语气选择
- ⚡ **威压**：施加压力，适合对付心理防线较弱的嫌疑人
- ☕ **温和**：建立信任，适合触及嫌疑人内心敏感点
- 🎭 **欺骗**：虚张声势，适合瓦解团伙成员的默契
- 🔍 **逻辑**：理性分析，适合对付善于狡辩的嫌疑人

### 判定系统
- 每阶段有明确的审讯目标，达成后可选择"继续"推进剧情
- 轮次有限，误判会消耗机会，机会用尽则案件中断
- 成功突破所有阶段即可结案

## 技术栈

- 原生 HTML5 / CSS3 / JavaScript（ES5）
- SVG 动态立绘生成
- Web Speech API 语音识别
- LocalStorage 进度存储
