# 《审讯室》角色设计与叙事深度迭代建议

> 分析角度：互动叙事与角色设计
> 分析对象：当前 3 个 Demo 角色 vs. 创意方案愿景
> 技术约束：纯前端、关键词匹配驱动、JSON 数据配置

---

## 一、当前 3 个角色的叙事深度与差异化评估

### 1.1 角色档案速览

| 维度 | 沈墨（案件1） | 林婉（案件2） | 老 K（案件3） |
|------|-------------|-------------|-------------|
| **性格标签** | 傲慢型 | 紧张型 | 冷静型 |
| **犯罪类型** | 暴力犯罪（抢劫） | 经济犯罪（诈骗） | 暴力犯罪（故意杀人） |
| **前职业** | 前拳击手 | 刚毕业大专生 | 前国企下岗工人 |
| **核心软肋** | 女儿高考、拳馆心血 | 母亲重病、 dialysis 费用 | 儿子车祸去世、欠薪 |
| **道德灰色度** | 有江湖规矩，不算全恶 | 被迫入行，被害人同情 | 被欠薪逼急，激情杀人 |
| **对威压反应** | 硬碰硬，情绪升高 | 崩溃哭泣，情绪大幅波动 | 近乎免疫，情绪不变 |
| **对温和反应** | 软化，提及家人 | 极软化，情感决堤 | 提及儿子时动容 |
| **对欺骗反应** | 被同伙"招供"激怒 | 被"同伙先招"击溃 | 对证据类欺骗微动 |
| **对逻辑反应** | 辩解、反问 | 结巴、语无伦次 | 理性回应、反将一军 |

### 1.2 差异化评估：足够，但存在结构性短板

**已做到的差异化（值得保留）：**

- **情绪反应曲线完全不同**：沈墨是"高对抗型"（威压升高情绪），林婉是"高敏感型"（任何压力都升高情绪，只有温和能降低），老 K 是"低反应型"（情绪波动极小，需精准打击软肋）。三条曲线形状各异，玩家能明显感受到"这个人跟上一个不一样"。
- **口语风格差异明显**：沈墨带江湖味（"老子""兄弟"），林婉带怯懦和哭腔（"我……我真的……"），老 K 带平淡的疲惫感（"我这条命早就不值钱了"）。
- **犯罪动机维度不同**： greed（贪婪/生计）、desperation（绝望）、grief（丧子之痛），覆盖了三种不同的犯罪心理底层。

**存在的结构性短板（需要迭代）：**

1. **家庭软肋高度同质化**：三个角色的"致命弱点"全部是家庭关系——女儿、母亲、儿子。这会让玩家在打通第二个角色后形成路径依赖："不管你是谁，聊家人就对了。" 家庭牌应该继续存在，但不应是每个角色的唯一解。
2. **暴力犯罪占比过高**：3 个角色中有 2 个是暴力犯罪（抢劫+杀人），与经济犯罪形成 2:1。创意方案中 6 种角色类型的犯罪类型分布应该更均衡，以便展现"审讯"这一行为的普适性——不仅是抓杀人犯，也包括瓦解白领犯罪、职务犯罪等。
3. **社会议题深度浅尝辄止**：
   - 沈墨的"下岗后江湖生存"只提了一句，没有深入。
   - 林婉的"杀猪盘底层员工"身份有挖掘空间，但当前仅停留在"我妈需要透析"，缺乏对诈骗产业链的揭示。
   - 老 K 的"国企下岗+欠薪杀人"是最有社会议题感的设计，但"二十万工钱"的数字对当代玩家来说冲击力不足，需要更具时代感的锚点。
4. **性格标签与行为逻辑偶有脱节**：
   - 沈墨作为"傲慢型"，在阶段 3 面对"钱"时却表现得"坐立不安"——这更像贪婪或恐惧，而非傲慢。傲慢型应该即使面对证据也保持姿态，直到最后才崩。
   - 林婉的"紧张型"在阶段 2 和 3 有向"配合型"过度滑动的趋势，缺乏"紧张型"独有的反复和自相矛盾（比如刚刚承认，下一秒又反悔）。

## 二、纯前端关键词匹配约束下，最大化"性格表现力"的策略

### 2.1 当前引擎的约束与机会

当前引擎逻辑（`engine.js`）：
```
1. 玩家选择 tone（4 选 1）
2. 玩家输入文本
3. 遍历 responseRules：先匹配 tone 一致，再匹配关键词包含
4. 命中则返回对应 reply + moodDelta
5. 未命中则从 fallback[high/mid/low] 中随机抽取
```

**约束**：没有 NLP 语义理解、没有上下文记忆（除了 mood 数值）、没有 LLM 生成能力。
**机会**：规则是纯 JSON，可快速扩展；mood 状态机提供了最基础的情绪分层；tone 系统天然提供了"策略选择"的玩法深度。

### 2.2 六大增强策略（全部可在当前技术栈内实现）

#### 策略 A：引入 "Tone 交叉反应" 机制

当前规则是 "Tone + Keyword" 的 AND 匹配。可以新增 "Tone 不匹配但 Keyword 命中" 的交叉规则，让角色对"错误策略"有鲜明反应。

**示例（沈墨阶段 1）：**
```json
// 当前：只有 intimidate + "指使" 才有专属回复
// 新增：friendly + "指使" 时，沈墨觉得被轻视
{
  "tone": "friendly",
  "matchKeywords": ["指使", "主谋", "策划"],
  "moodDelta": 8,
  "reply": "（冷笑）警官，你这种语气是在跟我聊天吗？我问你，你也配跟我聊这个？"
}
```

**示例（林婉阶段 2）：**
```json
// 当前：intimidate + "包庇" 让她恐惧
// 新增：deceive + "包庇" 时，她因为紧张而无法分辨真假
{
  "tone": "deceive",
  "matchKeywords": ["包庇", "窝藏", "隐瞒"],
  "moodDelta": 15,
  "reply": "（慌乱）我没有包庇！我真的没有！……等等，你们是不是在诈我？"
}
```

**实现成本**：每个阶段新增 3-5 条交叉规则，JSON 数据量 +30%。

#### 策略 B：情绪阈值触发 "特殊行为事件"

当前 mood 只影响 fallback 池的选择和立绘表情。可以引入 "当 mood 从 X 区间跌入 Y 区间时，触发一次强制特殊反应"。

**示例：**
- 沈墨 mood 从 >70 骤降至 <40（一次性降幅 >30）：触发"拍桌"事件，跳过玩家下一轮输入，直接显示拍桌动画+台词。
- 林婉 mood 从 <40 骤降至 <20：触发"哭泣"事件，连续 2 轮只能使用温和语气。
- 老 K mood 跌破 30（首次）：触发"摘下眼镜擦泪"事件，解锁"儿子"关键词的特殊权重。

**实现成本**：在 `app.js` 的 `updateMoodMeter()` 中增加阈值检测逻辑，无需改动引擎。

#### 策略 C：Fallback 池再分层——加入"关键词近 miss"回复

当前未命中规则时直接进 fallback 随机池。可以新增一层：玩家输入没有精确命中任何 rule 的 keywords，但命中了某个"相关词表"，则给出带有角色性格的"擦边回复"。

**示例（老 K）：**
```json
"nearMissKeywords": ["为什么", "怎么回事", "原因"],
"nearMissReply": "（平静地看着你）你问为什么。这世上很多事，没有为什么。"
```

这比通用的 "……" 或 "我没什么可说的" 更能维持角色存在感。

#### 策略 D：语气选择后的"角色可见反应"

当前 tone 选择只有 UI 高亮。可以在玩家选择 tone 后、输入文字前，让角色对"你选择的语气"做出一个微反应。

**示例：**
- 对沈墨选择 intimidate：他冷笑一声，活动了一下手腕。
- 对林婉选择 intimidate：她缩了缩肩膀，不敢看你。
- 对老 K 选择 friendly：他微微抬眼，似乎有些意外。

这不需要改引擎，只需要在 tone 按钮的点击事件里追加一段角色台词。

#### 策略 E：引入 "口头禅 / 口癖" 系统

每个角色在 fallback 回复中有一定概率插入自己的口癖，强化辨识度。

| 角色 | 口癖/高频词 | 插入场景 |
|------|-----------|---------|
| 沈墨 | "老子""笑话""哈" | 高 mood 时的 fallback |
| 林婉 | "我……""真的""求求你" | 所有 fallback |
| 老 K | "（停顿）""算了""没什么" | 中 mood 时的 fallback |

#### 策略 F：阶段过渡的"时间跳跃叙事"

当前阶段过渡只有一句 `sceneIntro`。可以在阶段之间加入"时间跳跃"的叙事细节，强化审讯的持久战感。

## 三、Demo 级可快速增强的叙事元素（附具体文案）

### 3.1 角色专属开场 / 过场 / 结局文案

#### 沈墨 —— 专属强化

**案件开场（进入案件时，增加一段 CG 感文字）：**
> 【档案调取 · 沈墨】
> 拳击馆的沙袋还在晃。你站在拳馆门口，看着墙上"强身健体，以德为先"的标语。
> 这个标语是沈墨自己写的。三年前他因为"以德为先"四个字跟人动手，把人打进了 ICU。
> 现在他以另一种方式"动手"——1300 万，两支枪，两个重伤的押运员。
> 拳馆里还有几个孩子在等他回去上课。其中之一，是他女儿。

**阶段 1→2 过场：**
> 【次日 · 10:02】
> 沈墨的眼眶下有淤青——不是被你打的，是他在拘留室里一夜没睡，自己用拳头砸墙砸的。
> 你故意把伪造报告滑到他面前。他看了一眼，嘴角抽搐了一下。
> "你们这些小把戏……"他说，但声音没有昨天那么大了。

**阶段 2→3 过场：**
> 【次日 · 16:30】
> 他的银行账户冻结令躺在桌上。你注意到他的手指在抖——那是打了几十年沙袋的手，以前从不会抖。
> "拳馆的地租……"他突然说了一半，闭嘴了。
> 那是他第一次提到"拳馆"时，语气里没有了骄傲。

**Good End 专属结局：**
> 1300 万在集装箱里被找到时，已经有 200 万被雨水泡坏了。
> 沈墨站在押送车前，回头看了你一眼。
> "我女儿……高考那天，别让她看见。"
> 他顿了顿，"拳馆里那帮孩子，你帮我找个教练。就说我……出去打比赛了。"
> 警车消失在凌晨的街道上。拳馆的招牌还亮着，你走过去，把它关了。

**Bad End 专属结局：**
> 第三阶段的误判让沈墨抓住了你的漏洞。
> "警官，你太急了。"他在椅子里调整了一个舒服的姿势，"你们没有证据，没有口供，没有赃款。"
> "我有的是时间。"
> 十天后，因为证据不足，他被取保候审。回到拳馆的那天，他给所有学员免了一个月的费。
> 有人在街上看见他，说他笑得很开心。

#### 林婉 —— 专属强化

**案件开场：**
> 【档案调取 · 林婉】
> 她的工位上还放着一个保温杯，杯身上贴着"妈妈加油"。
> 你翻开她的入职表：毕业院校一栏写着某职业技术学院，期望薪资 4500。
> 她实际的"工作"，是在电话里叫一个 53 岁的退休教师"老公"。
> 那个老师给她转了三笔钱，一共 800 万。最后一笔转完后，她在工位上哭了。
> 同事以为她是被客户骂哭的。

**阶段 1→2 过场：**
> 【次日 · 09:00】
> 她今天化了妆——不是为了你，是因为拘留所规定必须拍照。
> 你提到"王总"两个字时，她的睫毛颤了一下。
> "他就是个……微信头像用游艇的那种人。"她说。
> 那是她第一次用贬义词形容他。

**阶段 2→3 过场：**
> 【次日 · 17:00】
> 你告诉她，那个退休教师在得知真相后，吞了半瓶安眠药。抢救过来了。
> 林婉盯着桌子看了很久。然后说："他……他给我发过一首诗。我以为是系统文案。"
> 她的妆花了。

**Good End 结局：**
> 王志刚在香港落马时，是通过林婉提供的账户信息定位的。
> 她在看守所里收到了一封信——是那个退休教师写的。信里说："我不恨你，我恨的是让你变成这样的人。"
> 林婉在信纸背面写了几行字，托律师转交。
> 你后来听说，那是一份完整的诈骗话术手册。她一个字一个字默写出来的。
> "我只知道这些。"她说，"但也许能让少一个人被骗。"

#### 老 K —— 专属强化

**案件开场：**
> 【档案调取 · 老 K】
> 赵先生的别墅区在城北，房价六万一平。老 K 三个月前来这里修过水管，日薪 200，包一顿午饭。
> 他修完水管后，赵先生以"工程不合格"为由，扣了他 20 万工程款。
> 20 万，是老 K 给儿子小凯攒了五年的手术费首付。
> 小凯没等到手术。一场车祸，带走了他，也带走了老 K 最后一点"算了"。

**阶段 1→2 过场：**
> 【次日 · 11:00】
> 法医把伤口照片推到他面前。老 K 盯着看了很久，然后伸手，把照片转了个方向。
> "这个位置……"他指了指照片，"我干水电的时候，经常在这种角度切 PVC 管。"
> 他抬起头，"你们是不是觉得，我在暗示什么？"

**阶段 2→3 过场：**
> 【次日 · 19:30】
> 你给了他一支烟。他不要。
> "小凯走之前，让我戒烟。"他说，"我戒了三年了。"
> 他把烟放在桌上，没再碰。
> "你们想知道为什么。"他说，"我知道你们想问什么。但我得先问你一件事——"
> "如果是你，你会怎么做？"

**Good End 结局：**
> 老 K 的判决下来那天，你没有去。
> 但你去了一趟城北的公墓。小凯的墓碑旁边，多了一个新碑——赵先生的。
> 碑前有一束已经干枯的白菊，还有一把美工刀，刀柄上缠着电工胶带。
> 你把刀带走了。走到公墓门口时，你回头看了一眼。
> 一只鸟落在老 K 儿子的碑上，停了一会儿，飞走了。

### 3.2 情绪变化时的特殊反应设计

#### 沈墨 —— "傲慢型" 情绪反应事件

| 触发条件 | 事件名称 | 表现 | 叙事效果 |
|---------|---------|------|---------|
| mood 从 >70 骤降至 <40 | **拍桌** | 立绘震动 + 文字"（猛地站起，铁椅哗啦作响）" | 打断玩家节奏，制造压迫感 |
| mood <30 且连续 2 轮未命中有效规则 | **沉默凝视** | 跳过玩家一轮输入，显示"沈墨盯着你，一言不发。十秒。二十秒。" | 利用沉默制造心理压力 |
| 阶段 3 且 mood <25 | **第一次示弱** | "（声音低了下去）拳馆……拳馆的地租还没交。" | 傲慢面具首次脱落 |

#### 林婉 —— "紧张型" 情绪反应事件

| 触发条件 | 事件名称 | 表现 | 叙事效果 |
|---------|---------|------|---------|
| mood >80（因 intimidate 骤升） | **哭泣崩溃** | 连续 2 轮只能使用 friendly tone，其他 tone 按钮灰显 | 模拟"情绪过载"，强制玩家切换策略 |
| mood <30 且输入包含"王总" | **语无伦次** | 回复变成碎片化短句："他……不，我不是说……他让我……" | 展现内心矛盾 |
| 阶段 3 首次触发 low fallback | **主动提问** | "我如果……如果把账户说出来，我妈的透析……能不被停吗？" | 将个人命运与游戏机制绑定 |

#### 老 K —— "冷静型" 情绪反应事件

| 触发条件 | 事件名称 | 表现 | 叙事效果 |
|---------|---------|------|---------|
| mood 首次跌破 50 | **摘眼镜** | 立绘变化（眼镜消失）+ "（摘下眼镜，用衣角慢慢擦）" | 标志性"防御解除"信号 |
| mood 首次跌破 30 | **提及小凯** | 不需要关键词触发，强制插入一句关于儿子的回忆 | 揭露真正的软肋 |
| 阶段 3 且输入包含"理解" | **反问玩家** | "如果是你，你会怎么做？" + 进入"玩家必须回答"的特殊轮次 | 打破第四面墙，提升沉浸感 |

### 3.3 角色间差异化的审讯策略建议（玩家引导层面）

可以在案件 briefing 或 hint 中，以"审讯笔记"的形式给出策略建议，强化"每个角色需要不同打法"的认知。

#### 沈墨 —— 审讯策略建议

> 【审讯笔记 · 沈墨】
> 1. **不要硬碰硬**：威压对沈墨的效果是双刃剑——他会更对抗，但也可能暴露更多情绪。如果想速战速决，威压可用，但要做好"情绪过山车"的准备。
> 2. **江湖规矩是他的操作系统**：用"讲义气""给兄弟留条路"这类话术，比"减刑"更有效。他不信法律，信规矩。
> 3. **家庭的牌要最后出**：女儿是他的底牌，不是开场牌。太早提女儿，他会觉得你在威胁他。
> 4. **欺骗是最好的突破口**：他对"同伙出卖"的敏感度远高于"证据确凿"。

#### 林婉 —— 审讯策略建议

> 【审讯笔记 · 林婉】
> 1. **温和是唯一安全的入口**： intimidate 会让她情绪崩溃到无法提供有效信息。先用 friendly 降低 mood，再切其他 tone。
> 2. **她渴望被当作"受害者"而非"罪犯"**：让她感觉到"你理解她也是被逼的"，比让她感觉到"你在审问她"更有效。
> 3. **母亲是她的锚点，也是她的底线**：提及母亲可以软化她，但威胁停掉母亲的治疗会让她彻底封闭。
> 4. **逻辑追问要循序渐进**：她脑子会乱，一次性问太多问题，她只会摇头。

#### 老 K —— 审讯策略建议

> 【审讯笔记 · 老 K】
> 1. **威压几乎无效**：恐吓刑期、死刑对他没有用。他的心理账户里，自己的命早就"注销"了。
> 2. **逻辑是唯一语言**：他吃"道理"这一套。把案件还原成"如果当时是另一个人，结果会不会不同"的逻辑推演，能让他开口。
> 3. **儿子是唯一入口，但需要用对方式**：直接问"你儿子"，他会防御。用"如果小凯还在，他希望你怎么做"，他会动容。
> 4. **他可能在"求死"与"求生"之间摇摆**：如果让他觉得"认罪能让儿子的死有意义"，他会配合。

## 四、新增角色提案：第 4 个 Demo 角色

### 4.1 为什么需要第 4 个角色？

当前 3 个角色的性格覆盖：傲慢、紧张、冷静。
创意方案 6 种类型：冷面型、暴躁型、**演技型**、**懦弱型**、**高智商型**、忠诚型。

**缺失的类型中，"演技型"是当前 Demo 最需要的补充**，理由：
1. 现有 3 个角色在"说谎"这件事上都不够强——沈墨不屑于说谎，林婉不会说谎，老 K 懒得说谎。一个"以说谎为武器"的角色会彻底改变玩家的审讯体验。
2. 演技型天然与"欺骗" tone 形成对抗关系（你骗我？我演你。），能扩展 tone 系统的策略深度。
3. 经济犯罪/白领犯罪在当前 Demo 中占比偏低（仅林婉 1 个），需要补充非暴力犯罪。

### 4.2 角色设定：周牧野 —— "演技型 / 高智商型" 复合

| 属性 | 设定 |
|------|------|
| **姓名** | 周牧野 |
| **年龄** | 34 岁 |
| **表面身份** | 知名广告公司创意总监，曾获行业大奖，社交媒体上有 20 万粉丝 |
| **真实罪名** | 职务侵占 + 洗钱，通过虚构供应商合同，三年侵占公司 2600 万 |
| **性格标签** | **演技型**（主导）+ **高智商型**（辅助） |
| **核心特征** | 每一句回答都经过"人设管理"——他会根据你的 tone 切换自己的"表演人格"：面对威压时装受害者，面对温和时装合作者，面对逻辑时装专业人士，面对欺骗时……他会比你更会演。 |
| **犯罪动机** | 不是为钱——他账户里的钱几乎没动。是为了"证明系统可以被愚弄"的智力优越感。他曾在一个行业论坛上匿名发帖："广告的本质就是欺骗，而我是最优秀的广告人。" |
| **致命软肋** | 他的"作品"——那套完美的侵占方案。他无法忍受别人看不懂它的精妙，更无法忍受别人看懂了却不佩服。 |
| **审讯难点** | 他会主动"配合"你，但说的全是精心编排的半真半假。玩家面临的核心挑战不是"让他开口"，而是"分辨他说的哪句是真的"。 |

### 4.3 周牧野 3 阶段目标设计

#### 阶段 1：承认虚构供应商的存在

- **目标**：让周牧野承认"供应商公司是他虚构/控制的"
- **陷阱**：他可能会承认"供应商有问题"但把锅推给采购部同事
- **关键机制**：他会主动提供大量信息——但其中混杂真假。玩家需要用"逻辑" tone 追问细节，用"欺骗" tone 制造信息矛盾，才能逼他承认。
- **开场台词**：
  > "警官，我理解你们的工作。作为公民，我一定全力配合。不过我得提醒你们——我手上还有三个客户的比稿，如果因为我被冤枉而丢单，这个损失谁来承担？"

#### 阶段 2：承认资金最终流入他本人账户

- **目标**：确认侵占资金流向——通过 3 层空壳公司最终回到周牧野控制的离岸账户
- **陷阱**：他承认"资金流向有问题"，但声称是"行业惯例的返点操作"
- **关键机制**：这是"演技型"角色最精彩的部分——他会"表演认罪"，甚至流泪、忏悔，但说的全是编好的剧本。玩家如果此时误判"目标达成"，会触发 Bad End（陷阱判定）。
- **标志性台词**：
  > （深呼吸，眼眶微红）"我承认……供应商的选择我确实插手了。但那是为了公司利益！那几家供应商的价格比市场价低 15%——" 
  > （停顿，看向你）"哦，你们已经查过价格了？那你们应该知道，我没有说谎。"

#### 阶段 3：承认犯罪动机是"智力游戏"而非经济利益

- **目标**：套出他真正的动机——不是为了钱，而是为了证明"没有人能发现"
- **陷阱**：他会试图把动机包装成"职场压力""原生家庭问题"等更容易被同情的叙事
- **关键机制**：只有用"欺骗" tone（假装理解他的"创意"）或"逻辑" tone（精准拆解他的方案漏洞并表达"佩服"）才能触发真动机。威压和温和都会让他继续演。
- **真动机台词**：
  > （沉默了很久，突然笑了）
  > "你们知道吗？我那个方案，第三年才被审计发现漏洞。三年了，几百个专业人士，没人看出来。
  > 那 2600 万……我一分没花。我给我妈买了套房，写了她的名字，剩下的全在账户里。
  > 我不是为了钱。我是为了……证明我是个更好的骗子 than 你们所有人都是更好的侦探。"

### 4.4 周牧野对 4 种 Tone 的独特反应矩阵

| Tone | 他的反应模式 | 玩家策略 |
|------|-----------|---------|
| **威压** | 表演"受害者"——声音发抖、提及公司贡献、暗示要请律师和媒体 | 低效，他会把对话拖入程序性对抗 |
| **温和** | 表演"合作者"——积极配合，但提供的信息全部经过过滤 | 危险，容易误判为"目标达成" |
| **欺骗** | **兴奋**——终于有人陪他玩了。他会用更高明的谎言反击，但过程中会暴露破绽 | **高效**，但需要玩家能识破他的反骗 |
| **逻辑** | 表演"专业人士"——用行业术语和数据把你绕晕，但如果你真的跟上了，他会忍不住"纠正"你 | **高效**，他无法忍受别人"快要看懂了但还差一点点" |

## 五、当前对话规则数量评估与优化方案

### 5.1 当前规则数量评估

| 角色 | 阶段 1 | 阶段 2 | 阶段 3 | 总计 |
|------|--------|--------|--------|------|
| 沈墨 | 14 条 | 10 条 | 10 条 | 34 条 |
| 林婉 | 11 条 | 11 条 | 10 条 | 32 条 |
| 老 K | 10 条 | 9 条 | 10 条 | 29 条 |
| **总计** | | | | **95 条** |

**评估结论：数量处于"够用但紧绷"的临界点。**

- 每条规则覆盖 1-3 个关键词，在 8 轮限制下，玩家有较高概率触发至少 1 条规则。
- 但"未命中"的概率也不低——特别是当玩家使用同义词、变体表达时（比如"主谋"vs"幕后黑手"vs"老大"，当前只匹配了"主谋"）。
- 更关键的问题不是"数量"，而是"规则的结构设计"——当前规则设计存在以下可以优化的地方：

### 5.2 规则结构优化方案

#### 优化 1：关键词扩展——引入"同义词簇"

当前关键词是扁平数组。可以改为"关键词簇"结构，每个簇包含核心词+扩展词，扩展词命中时给予相同回复但轻微不同的 moodDelta。

**示例（沈墨阶段 1）：**
```json
{
  "tone": "intimidate",
  "matchClusters": [
    {
      "core": ["主谋", "策划", "组织者"],
      "extended": ["幕后黑手", "老大", "头目", "操盘"],
      "moodDelta": 5,
      "extendedMoodDelta": 3
    }
  ],
  "reply": "哈，指使？老子是那种被人指使的人吗？"
}
```

**效果**：玩家表达更自然时也能被理解，但用"专业术语"命中比"口语词"命中在情绪影响上更弱（更符合角色认知）。

#### 优化 2：引入 "连续命中奖励/惩罚" 机制

当前规则是"单次输入"匹配。可以增加一个轻量的"上下文"层：如果玩家连续 N 轮用同一 tone 且命中同一类关键词，触发角色的"疲劳/警觉"反应。

**示例：**
- 对沈墨连续 3 轮使用 intimidate：第 3 轮 moodDelta 翻倍（他更愤怒了）
- 对林婉连续 3 轮使用 friendly：第 3 轮触发"过度依赖"，她开始试探你是否真心
- 对老 K 连续 3 轮使用 logical：第 3 轮他"忍不住补充"，额外泄露信息

**实现**：在 `state` 中增加 `lastTone` 和 `consecutiveToneCount`，在 `generateReply` 前做预处理。

#### 优化 3：Fallback 池的"关键词权重"升级

当前 fallback 是完全随机的。可以改为：根据玩家输入与所有 rule keywords 的"相似度"，选择最接近的那条 fallback。

**简单实现方案（无需 NLP）：**
```javascript
// 在未命中任何规则时
var bestMatch = null;
var bestScore = 0;
rules.forEach(function(rule) {
  var score = 0;
  rule.matchKeywords.forEach(function(kw) {
    if (playerText.includes(kw)) score += 2;
    // 或者检查玩家输入的词与关键词是否有共同子串
    else if (hasCommonSubstring(playerText, kw, 2)) score += 1;
  });
  if (score > bestScore) { bestScore = score; bestMatch = rule; }
});

if (bestMatch && bestScore >= 2) {
  // 返回 bestMatch 的 reply，但 moodDelta 减半（因为不是精确命中）
} else {
  // 走原来的随机 fallback
}
```

这会大幅减少"玩家明明问得很对，但就是因为用了近义词而得到'……'"的挫败感。

#### 优化 4：规则数量建议

基于以上优化，推荐每个阶段的规则数量如下：

| 优化项 | 每阶段增量 | 说明 |
|--------|-----------|------|
| 核心规则（Tone + Keyword） | 12-16 条 | 保持当前密度 |
| 交叉反应规则（Tone 不匹配） | 3-5 条 | 新增 |
| 关键词簇扩展 | 将现有规则的 keywords ×2 | 结构优化，不增加规则条数 |
| 近 miss 回复 | 2-3 条 | 新增 |
| 阶段专属事件触发 | 1-2 条 | 新增（阈值触发类） |

**结论**：每个阶段从当前的 10-14 条增加到 **18-25 条**（含事件触发和交叉规则），总规则数从 95 条提升到约 **180 条**，可以在纯前端架构内实现，且能显著提升对话自然度。

## 附录：角色设计自检清单

为后续新增角色提供设计标准：

- [ ] **性格标签明确**：与现有角色不重复
- [ ] **犯罪类型差异化**：避免与已有角色类型过度重叠
- [ ] **软肋多元化**：不是只有"家人"一种设定
- [ ] **4 种 Tone 反应差异化**：每个 Tone 都有独特的反应模式
- [ ] **关键词覆盖自然**：口语、专业术语、俚语均有覆盖
- [ ] **有"反直觉"设计**：玩家第一印象与真实设定有落差，增加探索乐趣
- [ ] **社会议题锚点**：角色的故事能连接到某个真实的时代命题
- [ ] **有记忆点台词**：至少有一句玩家通关后还能记住的话
