# 《审讯室》游戏攻略

## 📖 游戏基础玩法

### 核心机制
- **目标**: 每个阶段有一个明确的审讯目标，需要让嫌疑人说出关键信息或你主动问出关键词
- **判断**: 在任意轮次点击「目标已达成」，系统会验证对话中是否出现目标关键词
- **失败机会**: 每阶段有 **3 次失败机会**（误判/漏判各消耗 1 次）
- **轮次限制**: 每阶段最多 **8 轮**，超过未达成目标则失败

### 四种语气效果

| 语气 | 图标 | 效果 | 适用对象 |
|---|---|---|---|
| **威压** ⚡ | 强硬施压 | 容易激怒嫌疑人，但能打破心理防线 | 傲慢型、冷静型 |
| **温和** ☕ | 共情沟通 | 降低抗拒度，更容易获取真实信息 | 紧张型、后悔型 |
| **欺骗** 🎭 | 虚张声势 | 制造假证据/假情报，诱使对方慌乱 | 傲慢型、紧张型 |
| **逻辑** 🔍 | 理性分析 | 用证据和逻辑推理施压 | 冷静型、所有类型 |

### 情绪系统（抗拒度 0-100）
- **高抗拒 (>70)**: 嫌疑人态度强硬，回答敷衍
- **中抗拒 (40-70)**: 态度松动，开始透露信息
- **低抗拒 (<40)**: 心理防线崩溃，容易坦白

## 🕵️ 案件攻略

### 案件 1：运钞车大劫案（CASE #2026-001）
**嫌疑人**: 沈墨（38岁，傲慢型）  
**类型**: 暴力犯罪  
**背景**: 前拳击手，性格傲慢但讲江湖规矩

#### 阶段 1：承认主谋身份
**目标**: 让沈墨承认自己是策划者  
**目标关键词**: 主谋、策划、组织者、指使、我策划、我组织的、我指挥、我是主谋

**攻略**:
1. **温和语气** + `家人/女儿/老婆` → 降低抗拒度（-15）
2. **欺骗语气** + `同伙已经招了/供出你` → 制造压力（-18）
3. **逻辑语气** + `策划/主谋/指使` → 直接质问
4. 当抗拒度低于 40 后，用 **温和语气** + `给个机会/认了` → 触发认罪

**推荐问话**:
- "沈先生，我们查到你女儿今年高考，你真的要因为这件事耽误她的未来吗？"（温和）
- "沈老板，你的同伙已经把你供出来了，你还想硬扛吗？"（欺骗）
- "说说吧，这起抢劫案到底是谁策划的？"（逻辑）

**目标**: 确认有 3 名同伙  
**目标关键词**: 三个、三个人、三人、两个人、两个同伙、还有两个、总共三个、我们三个

**攻略**:

3. **温和语气** + `兄弟/道上/规矩` → 触动江湖义气

**推荐问话**:
- "沈墨，你的兄弟已经扛不住了，他们说一共三个人参与，是不是？"（欺骗）
- "运钞车有两个押运员，一个人能搞定吗？"（逻辑）
- "江湖规矩我懂，但是你一个人扛得下来吗？"（温和）

## 💡 游戏技巧

### 通用策略
1. **先降低抗拒度**: 用温和语气聊嫌疑人的弱点（家人、过往经历）
2. **观察情绪变化**: 情绪条下降后再用逻辑或威压追问关键信息
3. **不要过早判断**: 确保对话中出现目标关键词后再点击"目标已达成"
4. **利用失败机会**: 如果不确定，可以先选"继续审讯"试探

### 角色性格应对
- **傲慢型**: 先用威压打破傲慢，再用欺骗制造恐慌，最后用温和给台阶
- **紧张型**: 全程用温和语气建立信任，避免威压刺激
- **冷静型**: 用逻辑和证据一步步施压，找到情感弱点突破

### 关键词触发技巧
- 对话中出现目标关键词即可达成目标，不一定需要嫌疑人亲口承认
- 你可以直接问出关键词（如"你是主谋吗？"），也算达成
- 注意陷阱关键词，避免误判
